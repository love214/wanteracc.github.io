粗略模仿微信UI

项目名称：WeChatUI

项目地址：Github

  • 粗略模仿微信UI

    切换fragment,在运行的过程中修改显示的fragment

    public void onClick(View v) {

    FragmentManager manager = getFragmentManager();
FragmentTransaction transaction=manager.beginTransaction();
switch (v.getId()) {
    case R.id.btn_contact:
        transaction.replace(R.id.ll, new ContactFragment());
        break;

    case R.id.btn_discover:
        transaction.replace(R.id.ll, new DiscoverFragment());
        break;

    case R.id.btn_wx:
        transaction.replace(R.id.ll, new WeXinFragment());
        break;

    case R.id.btn_me:
        transaction.replace(R.id.ll, new MeFragment());
        break;
}
transaction.commit();

    }

在HomeActivity布局文件中，定义了一个线性布局，R.id.ll，使用fragment动态替换该布局。
